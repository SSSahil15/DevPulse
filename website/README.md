# DevPulse Documentation Website

This directory contains the official documentation for DevPulse, built using [Docusaurus](https://docusaurus.io/).

## Features

- **Developer Guides**: Architecture, API references, and security compliance info.
- **User Guides**: Getting started, dashboard navigation, and best practices.

## Installation

```bash
npm install
```

## Local Development

```bash
npm run start
```

This command starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

## Build

```bash
npm run build
```

This command generates static content into the `build` directory and can be served using any static contents hosting service.
