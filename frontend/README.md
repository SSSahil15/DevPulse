# DevPulse Frontend

This directory contains the React & Vite frontend application for the DevPulse platform.

## Tech Stack

- **React 18** with **Vite**
- **Tailwind CSS** for styling (with Glassmorphism design elements)
- **Recharts** for visualizing the DevPulse Score and telemetry data
- **Socket.io Client** for real-time analysis progress updates
- **Lucide React** for icons

## Local Setup

1. Install dependencies: `npm install`
2. Create a `.env` file containing `VITE_API_URL=http://localhost:4000` (or your backend URL).
3. Start the development server: `npm run dev`
